# CODEEX AI - Android Guide

## Installing on Android

CODEEX AI is a Progressive Web App (PWA) that provides a native app-like experience on Android devices.

### Installation Steps

#### Method 1: Chrome Browser (Recommended)

1. Open Chrome on your Android device
2. Navigate to your CODEEX AI URL
3. Wait for the page to fully load
4. Tap the **menu icon** (⋮) in the top right
5. Select **"Add to Home screen"** or **"Install app"**
6. Tap **"Install"** in the confirmation dialog
7. The app icon appears on your home screen

#### Method 2: Samsung Internet

1. Open Samsung Internet browser
2. Navigate to your CODEEX AI URL
3. Tap the **menu icon** (≡)
4. Select **"Add page to"** → **"Home screen"**
5. Tap **"Add"**

#### Method 3: Firefox

1. Open Firefox on Android
2. Navigate to your CODEEX AI URL
3. Tap the **menu icon** (⋮)
4. Select **"Install"**
5. Confirm installation

## Troubleshooting

### App Won't Install

**Solution:**
1. Clear Chrome cache and data
2. Ensure you're using HTTPS
3. Try incognito mode first
4. Update Chrome to latest version

### App Crashes

**Solution:**
1. Clear app data: Settings → Apps → CODEEX AI → Clear Data
2. Reinstall the app
3. Check available storage space

### Voice Input Not Working

**Solution:**
1. Grant microphone permission
2. Check Google Voice Typing is enabled
3. Ensure internet connection
4. Try restarting the app

### Camera Not Working

**Solution:**
1. Grant camera permission
2. Close other apps using camera
3. Restart the app

### Offline Mode Issues

**Solution:**
1. Visit the app while online first
2. Wait for service worker to cache
3. Check storage isn't full
