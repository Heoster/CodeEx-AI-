# Product Overview

## Project Purpose
CODEEX AI is a comprehensive conversational AI platform built with Next.js that provides intelligent, context-aware interactions with advanced features including voice support, visual problem solving, and real-time web search capabilities. Created by heoster, it serves as a fully-functional AI assistant application showcasing real-world AI capabilities.

## Value Proposition
- **Intelligent Conversations**: Natural, context-aware dialogues with conversation history and adaptive responses
- **Multi-Modal Interaction**: Support for text, voice, and image-based inputs
- **Persistent Experience**: Automatic conversation saving with multi-chat management
- **Real-Time Intelligence**: Web search integration for up-to-date information with citations
- **Secure & Reliable**: Firebase-powered authentication with email/password and Google sign-in

## Key Features

### Core Capabilities
1. **Dynamic AI Conversations**: Context-aware dialogues that remember history and adapt tone/technical level
2. **Multi-Chat Management**: Create and manage multiple simultaneous chat sessions for different topics
3. **Persistent Chat History**: Automatic local storage of conversations for seamless continuation
4. **Visual Problem Solver**: Upload images of handwritten math equations for recognition and step-by-step solutions
5. **Integrated Command System**: Powerful slash commands (`/solve`, `/summarize`, `/search`)
6. **Real-Time Web Search**: Browse the web for current information with grounded responses and citations
7. **Voice-Enabled Interaction**: Full voice support with microphone input and speech output
8. **Secure Authentication**: Firebase-based user authentication with multiple sign-in methods

### User Experience Features
- **Progressive Web App (PWA)**: Installable app with offline capabilities
- **Theme Support**: Light and dark mode toggle for user preference
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Content Customization**: Adjustable output tone, profanity filter, and technical level
- **Guided Tour**: Interactive app tour for new users
- **PDF Analysis**: Dedicated PDF analyzer functionality

## Target Users
- Students seeking homework help and problem-solving assistance
- Professionals requiring quick information retrieval and summarization
- Developers looking for code assistance and technical explanations
- General users wanting an intelligent conversational AI assistant
- Anyone needing multi-modal (text, voice, image) AI interaction

## Use Cases
- Solving mathematical equations from images
- Managing multiple research topics in separate chat sessions
- Voice-based hands-free interaction
- Real-time web search for current events and information
- Document analysis and summarization
- Technical problem-solving with adjustable complexity levels
- Secure personal AI assistant with conversation history
